Never Miss a Moment

During games the team manager creates game events e.g. Progress updates (Kick Off, Half Time, Full Time), Score updates (Our Team 2 | 1 The Opposition), and more with the ability to capture individual game event notes. These can be delivered instantly to your phone. You're in control! By choosing what you want to be notified about, you can: get notifications about all events, just progress events, scores only or turn notifications off entirely AND you can fine-tune competition alerts separately. You're always in control and in the loop! ...AND if you want to see any details for any game (Live or Completed) - just open WhatScore!

Your Team, Your Rules

When creating a team (or competition) a unique share code is generated. Fans can use that code to find and join the team or comp. If a team/comp is public, anyone with the code can follow instantly and start receiving live updates. With private teams/comps, join requests need to be approved by the team/comp owner or manager. Owners/Managers can regenerate new codes at any time. Either way, you stay in control.
